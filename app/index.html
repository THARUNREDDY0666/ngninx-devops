About This Project

This project demonstrates a full DevOps workflow using Nginx, Docker, and Kubernetes. It includes automated deployment, scaling, and service exposure using NodePort. The page you see is served by an Nginx container running in a Minikube cluster.

Project Features

Docker Container

The app is containerized using Docker. The image is built and run inside the Minikube environment to avoid ImagePullBackOff issues.

Kubernetes Deployment

Deployment uses a Kubernetes Deployment object with 2 replicas, ensuring high availability and automated restarts.

Kubernetes Service

A NodePort Service exposes the app externally, allowing access through the Minikube IP and the assigned port.
